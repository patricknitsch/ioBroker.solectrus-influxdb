What this tab shows

This tab provides a real-time overview of all active sensors and computed values configured in this adapter instance. It refreshes automatically every 5 seconds.

InfluxDB Sensors

Active sensors are displayed as compact cards in a responsive grid — all sensors are visible at once without scrolling. Each card shows:

  • Sensor name and a type badge (int, float, bool, string, json) in the header
  • Current value — live reading; n/a if no value has been received yet
  • Measurement · field — the target location in InfluxDB
  • Source state — the ioBroker state ID being read (truncated, full path shown on hover)

Formula Engine table

Shown only when Data-SOLECTRUS is enabled in the adapter settings. Lists all active computed items with their current values, mode (Formula, Source, or State Machine), and expression.

Navigation

Click Open Configuration in the toolbar to jump directly to the adapter settings where you can add or edit sensors.

Was dieser Tab anzeigt

Dieser Tab zeigt eine Echtzeit-Übersicht aller aktiven Sensoren und berechneten Werte dieser Adapterinstanz. Er aktualisiert sich automatisch alle 5 Sekunden.

InfluxDB-Sensoren

Aktive Sensoren werden als kompakte Karten in einem responsiven Raster angezeigt — alle Sensoren sind auf einen Blick sichtbar, ohne zu scrollen. Jede Karte zeigt:

  • Sensorname und ein Typ-Badge (int, float, bool, string, json) im Header
  • Aktueller Wert — Live-Messwert; k.A. wenn noch kein Wert empfangen wurde
  • Measurement · Field — das Ziel in InfluxDB
  • Quell-State — die gelesene ioBroker-State-ID (gekürzt, voller Pfad als Tooltip)

Formel-Engine Tabelle

Wird nur angezeigt, wenn Data-SOLECTRUS in den Adapter-Einstellungen aktiviert ist. Listet alle aktiven berechneten Items mit aktuellen Werten, Modus (Formel, Quelle oder State Machine) und Ausdruck auf.

Navigation

Auf Konfiguration öffnen in der Toolbar klicken, um direkt zu den Adapter-Einstellungen zu gelangen, wo Sensoren hinzugefügt oder bearbeitet werden können.
